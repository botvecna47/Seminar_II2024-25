12-Week Learning Roadmap

Week 1-2: Python Fundamentals

  • Python Introduction & Setup
  • Python Syntax & Comments
  • Variables & Data Types
  • Operators
  • Strings & Formatting
Easy
  1. Circle Area Calculator
    Problem Statement:

    Write a Python program that calculates the area of a circle given its radius. The program should:

    • Take the radius as input from the user
    • Validate that the input is a positive number
    • Calculate the area using the formula: πr²
    • Display the result rounded to 2 decimal places
    Sample Input:
    Enter the radius: 5
    Sample Output:
    The area of the circle is: 78.54 square units
  2. Temperature Converter
    Problem Statement:

    Create a temperature conversion program that converts between Celsius and Fahrenheit. The program should:

    • Allow users to choose the conversion direction (C to F or F to C)
    • Accept the temperature value as input
    • Implement the conversion formulas:
      • Celsius to Fahrenheit: (C × 9/5) + 32
      • Fahrenheit to Celsius: (F - 32) × 5/9
    • Display the result with appropriate units
    Sample Input:
    Enter temperature: 32
Choose conversion (1: C to F, 2: F to C): 1
    Sample Output:
    32°C is equal to 89.6°F
Medium
  1. Character Frequency Counter
    Problem Statement:

    Develop a program that analyzes the frequency of characters in a given string. The program should:

    • Accept a string input from the user
    • Count the occurrence of each character (case-sensitive)
    • Ignore spaces and special characters
    • Display the results in descending order of frequency
    • Handle empty strings and invalid inputs appropriately
    Sample Input:
    Enter a string: Hello World!
    Sample Output:
    Character frequencies:
l: 3
o: 2
H: 1
e: 1
W: 1
r: 1
d: 1
  2. Simple Calculator
    Problem Statement:

    Implement a calculator that can perform basic arithmetic operations. The program should:

    • Support addition, subtraction, multiplication, and division
    • Accept two numbers and an operator as input
    • Handle division by zero and invalid inputs
    • Display the result with appropriate formatting
    • Allow continuous calculations until the user chooses to exit
    Sample Input:
    Enter first number: 10
Enter operator (+, -, *, /): *
Enter second number: 5
    Sample Output:
    Result: 10 * 5 = 50
Hard
  1. Password Generator
    Problem Statement:

    Create a secure password generator with specific requirements. The program should:

    • Generate passwords with the following criteria:
      • Minimum length of 8 characters
      • At least one uppercase letter
      • At least one lowercase letter
      • At least one number
      • At least one special character
    • Allow users to specify password length
    • Provide an option to exclude similar characters
    • Include a password strength indicator
    • Allow copying the generated password to clipboard

Password copied to clipboard!
