Week 1-2: Python Fundamentals
  • Python Introduction & Setup
  • Python Syntax & Comments
  • Variables & Data Types
  • Operators
  • Strings & Formatting
Easy
  1. Circle Area Calculator
    Problem Statement:

    Write a Python program that calculates the area of a circle given its radius. The program should:

    • Take the radius as input from the user
    • Validate that the input is a positive number
    • Calculate the area using the formula: πr²
    • Display the result rounded to 2 decimal places
    Sample Input:
    Enter the radius: 5
    Sample Output:
    The area of the circle is: 78.54 square units
  2. Temperature Converter
    Problem Statement:

    Create a temperature conversion program that converts between Celsius and Fahrenheit. The program should:

    • Allow users to choose the conversion direction (C to F or F to C)
    • Accept the temperature value as input
    • Implement the conversion formulas:
      • Celsius to Fahrenheit: (C × 9/5) + 32
      • Fahrenheit to Celsius: (F - 32) × 5/9
    • Display the result with appropriate units
    Sample Input:
    Enter temperature: 32
Choose conversion (1: C to F, 2: F to C): 1
    Sample Output:
    32°C is equal to 89.6°F

  2. Simple Calculator

    Implement a calculator that can perform basic arithmetic operations. The program should:

    • Support addition, subtraction, multiplication, and division

Week 12: Final Project & Review
Final Project

Capstone Challenge: Build a complete Python application that demonstrates your understanding of the topics covered in this roadmap. Choose a project idea that excites you, or pick from the suggestions below:

  • Student Management System (with file/database storage, CRUD, search, and reporting)
  • Personal Expense Tracker (with data visualization and export features)
  • Quiz or Flashcard App (with user scores and question banks)
  • File Organizer/Manager (automate file sorting, renaming, and backup)

  1. To-Do List CLI App
    Problem Statement:

    Build a command-line To-Do List application in Python. The app should allow users to add, view, and mark tasks as complete.

    • Display a menu with options: Add Task, View Tasks, Mark Complete, Exit
    • Store tasks in a list (in memory is fine for this exercise)
    • Show completed/incomplete status for each task
    • Handle invalid menu choices gracefully
    Sample Input:
    1. Add Task
2. View Tasks
3. Mark Complete
4. Exit
Enter choice: 1
Enter task: Buy groceries
    Sample Output:
    Task added!
1. [ ] Buy groceries
2. [ ] Finish assignment
Enter choice: 3
Enter task number to mark complete: 1
Task marked as complete!
1. [x] Buy groceries
2. [ ] Finish assignment

  1. Personal Expense Tracker
    Problem Statement:

    Create a Python program that tracks personal expenses and generates a summary report.

    • Allow users to add expenses with amount, category, and date
    • Store expenses in a list or file (CSV/JSON)
    • Display a summary: total spent, spending by category, and most expensive item
    • Provide an option to export the report to a file
    Sample Input:
    Add expense: 500, Groceries, 2024-06-01
Add expense: 1200, Rent, 2024-06-01
Add expense: 200, Transport, 2024-06-02
    Sample Output:
    Expense Summary:
Total spent: 1900
By category:
  Groceries: 500
  Rent: 1200
  Transport: 200
Most expensive: Rent (1200)
Report exported to expenses_report.txt
Hard
  1. Student Management System (Mini Project)
    Problem Statement:

    Develop a Student Management System in Python that allows users to manage student records.

    • Add, view, update, and delete student records (CRUD)
    • Each record should have: ID, Name, Age, Grade, Courses (list)
    • Support searching students by name or ID
    • Store data in a file (CSV/JSON) for persistence
    • Display a summary report: total students, average age, grade distribution
    Sample Input:
    Add student: 1001, John Doe, 20, A, [Math, Science]
Add student: 1002, Alice Smith, 19, B, [English, History]
Search by name: John
    Sample Output:
    Student Records:
ID: 1001, Name: John Doe, Age: 20, Grade: A, Courses: Math, Science
ID: 1002, Name: Alice Smith, Age: 19, Grade: B, Courses: English, History
Summary:
Total students: 2
Average age: 19.5
Grade distribution: A: 1, B: 1
